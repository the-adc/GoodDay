Создано UMGEEK.


Используйте L + I, чтобы карабкаться вверх и вниз по определенной лестнице


				Коды

OLDLDD = Изменить тип анимации при подъеме и спуске по лестнице

TELELDD = Телепортируется на последнюю добавленную лестницу

ENDLDD = "конец мода", если вы используете этот чит, вы не сможете подниматься 
по лестнице, пока не перезапустите игру




Файл Ladders v2.2.ini 
Этот файл имеет настройки и координаты лестницы в моде, это важно для работы 
мода. 


Здесь объясняются функции 
INI

KEY1 и KEY2 - это клавиши, которые вы должны комбинировать, чтобы идти вверх и 
вниз по лестнице, подходить к лестнице и нажимать 
две клавиши вместе, 
первоначально это будут клавиши L и I, но их можно изменить с помощью этой 
страницы, поместив значение ключа в KEY1 и KEY2.
READ_SPEED, показывает, когда мод ищет лестницы в INI-файле, если он находится 
здесь в 0, ничего не будет отображаться, и это будет быстрее.
 
SHOW_LADDER_NUM, показывает номер каждой лестницы во время чтения, если он 
равен 0, он ничего не показывает.

OLD_VERSION, изменить анимацию при переходе вверх и вниз по лестнице с
 помощью 1 и 0, можно изменить в диапазоне с помощью чит-кода OLDLDD, 
однако это
 не записывается в INI и при запуске с начала стоит того, что здесь есть.

COUNT, продвинутая часть. изменить только при добавлении новой лестницы. в
будущем, если кому-то будет интересно, я могу объяснить,
 как добавить лестницу в
мод.

RADIUS, он определяет расстояние по лестнице, чтобы можно было использовать
команду для набора высоты, чем меньше 
значение, тем лучше визуальное
изображение и сложнее подниматься.

SPEED_JUMP, установите скорость, которая будет идти вверх и вниз, это работает 
только с OLD_VERSION на 0
.
SPEED_ANIM, изменяет скорость анимации. Это работает только с OLD_VERSION на 
0.
